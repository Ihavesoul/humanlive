# Неделя

| День | Сессия | Время |
|---|---|---:|
| Пн | [[04_Full_Body_A]] | 50–60 мин |
| Вт | [[07_Scoliosis_Session_A]] | 40–50 мин |
| Ср | прогулка + дыхание | 20–40 мин |
| Чт | [[05_Full_Body_B]] | 50–60 мин |
| Пт | [[08_Scoliosis_Session_B]] | 45–55 мин |
| Сб | [[06_Full_Body_C]] | 50–60 мин |
| Вс | отдых/лёгкая ходьба | по самочувствию |

Дополнительно — набрать 150–300 минут умеренной аэробной активности в неделю. Для текущего инвентаря это в основном быстрая ходьба. Не надо превращать пять тренировочных дней в пять тяжёлых дней: motor-control и ходьба должны восстанавливать, а не конкурировать с силовыми.

# Правило симметрии

Односторонние упражнения выполняются с одинаковым числом подходов и целевым усилием. Если одна сторона технически сдаётся раньше, завершить её при целевом RIR и не заставлять догонять повторы скручиванием. Вторая сторона выполняет сопоставимое усилие, но не получает автоматически лишние подходы.

# Минимум, который считается выполненной тренировкой

В плохой день:

1. разминка;
2. одно упражнение на ноги;
3. одно push;
4. одно pull;
5. одно trunk-control;
6. 1–2 подхода каждого при RIR 3–4.

Это лучше, чем пропускать неделю и затем компенсировать двойным объёмом.
