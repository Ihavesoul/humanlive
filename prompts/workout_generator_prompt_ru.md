# Workout Generator Prompt v1.0.0

Сгенерируй программу, строго соблюдая system policy и JSON Schema `workout_response.schema.json`.

## Вход

```json
{{WORKOUT_REQUEST_JSON}}
```

## Задача

1. Проверь red flag gate.
2. Сформируй программу на `weeks` недель с указанным числом силовых и motor-control сессий.
3. Используй только разрешённые упражнения и имеющийся инвентарь.
4. Уложи каждую сессию в лимит времени.
5. Для односторонних упражнений укажи равные подходы на стороны, пока side-specific content заблокирован.
6. Добавь прогрессию, deload и регрессии.
7. Включи отдельные предупреждения:
   - BIA не является точным измерением состава тела;
   - общая тренировка не является curve-specific PSSE;
   - side-specific блок остаётся закрыт без clinician data.
8. Укажи все существенные неопределённости.
9. Верни JSON без дополнительного текста.

## Недопустимые упрощения

- Не выводи «качайте слабую сторону» без клинической классификации.
- Не делай вывод о стороне укорочения/гипертрофии из субъективного ощущения.
- Не заменяй тяговые движения только упражнениями на лопатки: в каждой силовой неделе должен быть горизонтальный pull.
- Не добавляй инвентарь, которого нет.
- Не выдавай конкретный диагноз по протрузиям/узлам Шморля.
