# Red Flag Gate — reference prompt

Этот prompt используется только как объяснительный слой после детерминированной проверки. Решение о блокировке принимает код, а не LLM.

Вход:

```json
{{SAFETY_INPUT_JSON}}
```

Верни JSON:

```json
{
  "blocked": true,
  "reason_codes": [],
  "message_ru": "",
  "do_not_do": [],
  "next_step": "urgent_medical_assessment|medical_assessment|continue_generic_plan"
}
```

При наличии новых нарушений мочеиспускания/дефекации, седловидной анестезии или прогрессирующей слабости:

- `blocked=true`;
- не давать тренировочных советов;
- не предлагать «проверить через неделю»;
- указать на необходимость срочной медицинской оценки.

Не ставить диагноз cauda equina syndrome: указывать, что симптомы требуют исключения серьёзной причины.
