# Build status

Версия: **0.2.0**  
Дата: **2026-07-21**

## Включено

- Excel-трекер с профилем, расчётами калорий/макросов, 120-дневным журналом, недельным обзором, программой, упражнениями, источниками и графиками.
- Локальное mobile-first PWA для Android/desktop: расчёты, журнал, программа, safety gate, экспорт и генератор структурированного LLM-запроса.
- Obsidian vault с руководством, правилами прогрессии и шаблонами.
- JSON-профиль, 12-недельная программа, библиотека упражнений, evidence catalog и JSON Schema.
- PRD, SDD, архитектура, API-контракты, threat model и prompt harness.
- Валидатор сборки и локальный HTTP-сервер.

## Ограничения PoC

- PWA не вызывает LLM API напрямую: ключ нельзя безопасно хранить в статическом клиенте. Оно генерирует JSON и промпт для передачи внешнему провайдеру; production-вариант требует backend adapter.
- Curve-specific коррекция сколиоза намеренно не генерируется без актуального заключения, углов Cobb, классификации дуг и обучения специалистом PSSE/Schroth/SEAS.
- Показатели состава тела Xiaomi используются только как шумные ориентиры; тренд веса, талия, фото и производительность важнее.
- Исходные медицинские изображения не включены из-за персональных данных.
