# PWA

Из корня проекта:

```bash
python scripts/serve_pwa.py
```

Откройте напечатанный URL. Для телефона компьютер и Android должны находиться в одной локальной сети. Профиль и журнал хранятся в `localStorage`; используйте JSON export для резервной копии. Приложение не содержит API-ключей: раздел LLM генерирует структурированный запрос для backend/provider integration.
